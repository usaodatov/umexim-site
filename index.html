UMEXIM.COM - E-Commerce Client-Server Demo

This SWE4303 Computing Infrastructure project for Regent College London, University of Bolton, powers an eco-friendly e-commerce marketplace. The Azure-hosted frontend (umexim.com) connects securely via HTTPS to a Flask backend running on an Azure VM (api.umexim.com), with DNS routing managed through AWS Route 53.

Test the system:
- /api/ping checks backend connectivity.
- /api/message mimics product queries or transactions.
