Project Description

This website was built as part of the SWE4303 Computing Infrastructure module.
It demonstrates frontend-backend integration hosted via Azure Static Web Apps.
The frontend interacts with a Python Flask backend deployed on an Ubuntu VM.

System Setup

  • Frontend: HTML/CSS, deployed on Azure Static Web App
  • Backend: Python Flask, hosted on Ubuntu VM
  • Connected via HTTPS requests (secured NGINX proxy)

Project Features

  • API ping check
  • Message exchange using POST
  • Live deployment: https://umexim.com
